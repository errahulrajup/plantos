# Release Checklist

This checklist must be completely satisfied before any major version bump is permitted.

- [ ] Architecture Frozen (No new core engines).
- [ ] RFCs Updated & Approved.
- [ ] ADRs Updated & Approved.
- [ ] Unit & Integration Tests Pass.
- [ ] Configuration Compiler Pass.
- [ ] Demo Plant Pass (Simulation).
- [ ] PlantSmör Reference Package Pass (Validation).
- [ ] Documentation Complete.
- [ ] Performance Baseline Recorded in KPIs.
- [ ] Zero Critical Architecture Violations (Fitness Tests pass).
- [ ] "One-click" Local Setup Verified.
